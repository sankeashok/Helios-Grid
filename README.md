# ☀️ Helios-Grid: Enterprise Energy Consumption MLOps Pipeline

🚀 **Live Demo**: Production-grade MLOps pipeline for energy consumption prediction

## 🎯 Features
- **Real-time Energy Predictions** using XGBoost & LightGBM
- **Interactive API Documentation** with Swagger UI
- **Enterprise CI/CD Pipeline** with automated testing
- **Premium Dashboard** with glassmorphism design
- **Comprehensive MLOps** with model tracking & monitoring

## 🔗 Quick Links
- **API Documentation**: Click "Open in Browser" then go to `/docs`
- **Health Check**: `/health`
- **Energy Prediction**: `/predict`
- **Model Info**: `/model/info`

## 🛠️ Technology Stack
- **ML**: XGBoost, LightGBM, scikit-learn
- **API**: FastAPI, Pydantic
- **Data**: Kaggle Hourly Energy Consumption Dataset
- **DevOps**: Docker, GitHub Actions, Enterprise CI/CD
- **Monitoring**: MLflow, Prometheus integration ready

## 📊 Dataset
Using [Hourly Energy Consumption Dataset](https://www.kaggle.com/datasets/robikscube/hourly-energy-consumption/data) by [@robikscube](https://www.kaggle.com/robikscube) - 10+ years of PJM Interconnection LLC energy data.

## 🚀 Usage
1. Click "Open in Browser" above
2. Visit `/docs` for interactive API documentation
3. Use `/predict` endpoint for energy consumption predictions
4. Check `/health` for system status
5. Explore the premium dashboard interface

## 🏆 Enterprise Features
- 7-Layer CI/CD Pipeline Architecture
- Multi-stage Backend Testing (Python 3.9+)
- React Frontend Testing & Build
- Docker Containerization with Security Scanning
- Automated Deployment Simulation
- Comprehensive Wiki Documentation

Built with ❤️ for enterprise energy analytics and sustainable grid management.
